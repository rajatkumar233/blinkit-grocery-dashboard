# Blinkit Grocery Data Dashboard

This project is about studying Blinkit grocery store data to find useful patterns about what products sell the most, how stores compare, and what customers like.

## About This Project

I looked at more than 8,500 rows of real Blinkit grocery data. The goal was to understand things like:

- Which product types sell the most
- How the size of a store changes its sales
- Which locations do better than others
- What customers rate highest and lowest
- How fat content of products affects sales

## Tools I Used

- **Microsoft Excel** — For cleaning and organizing the raw data
- **Power BI** — For building the interactive dashboard with charts and filters

## What I Found

Here are some interesting things from the analysis:

- Fruits, Vegetables, and Snacks are the top selling categories
- Medium sized stores make more sales than small or big ones
- Tier 3 cities have surprisingly strong sales numbers
- Low fat products sell slightly more than regular fat ones
- Customer ratings are mostly between 3.5 and 4.0 across all stores

## How to Use

1. Download the `.pbix` files from this repo
2. Open them in **Power BI Desktop** (free from Microsoft)
3. The dashboard will load with all the charts and filters ready to explore
4. You can also open the CSV file in Excel to look at the raw data
